## ðŸ§ª How to Run Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/nyayai.git
   cd nyayai
   ```

2. **Setup Backend**

   ```bash
   cd server
   npm install
   npm run dev
   ```

3. **Setup Frontend**

   ```bash
   cd client
   npm install
   npm start
   ```

4. **Environment Variables**
   Create a `.env` file in `/server` and `/client` directories with necessary keys:

   * `MONGODB_URI`
   * `GEMINI_API_KEY`
   * `JWT_SECRET`
